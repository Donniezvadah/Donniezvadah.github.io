# This is my personal website

Welcome to my personal website! This site is built using Hugo and the Academic CV theme.

## About This Site

This website showcases my professional background, projects, and research. It's built with Hugo, a fast and flexible static site generator, and uses the Academic CV theme for a clean, professional appearance.

## Technologies Used

- **Hugo**: Static site generator
- **Academic CV Theme**: Professional resume/cv theme
- **GitHub Pages**: Hosting
- **Markdown**: Content management
- **Git**: Version control
- **VS Code**: Development environment
- **Hugo Blox Builder**: Website building toolkit
- **Responsive Design**: Mobile-friendly layout
- **Continuous Deployment**: Automated updates
- **SEO Optimization**: Search engine friendly
- **Custom Domain**: Personalized URL
- **Fast Loading**: Optimized performance
- **Secure Hosting**: HTTPS enabled
- **Cross-browser Compatible**: Works on all modern browsers
- **Accessibility**: WCAG compliant design
- **Easy Maintenance**: Simple content updates
- **Version Controlled**: Git-based workflow
